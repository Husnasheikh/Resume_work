# Student Management System (Backend)

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Features
- Add student
- View all students
- View student by ID
- Update student details
- Delete student

## API Endpoints
POST /students  
GET /students  
GET /students/:id  
PUT /students/:id  
DELETE /students/:id  

## How to Run
1. npm install
2. Create .env file
3. Add MongoDB URI
4. node server.js
